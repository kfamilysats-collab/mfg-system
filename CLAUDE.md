## 2026/4/21 実装完了：見積依頼書ページ

### 実装済み機能
- sec-estimate ページとして製造管理システムに追加済み
- 横A4・モノクロ・FAX前提のレイアウト
- 印刷方式：別ウィンドウ（window.open）でCtrl+Pで印刷
- ファイル名：見積依頼書_会社名_日付時刻.html のタイトルで自動設定

### データ構造
- DB.estimates：見積依頼書データ
- DB.estimateTemplates：備考文言テンプレート
- 各estimate：{ id, number, date, createdAt, client（材料発注先）, requester（依頼先・受注先）, items[], selectedTemplateTexts[], extraNotes, status, faxSent }
- 各item：{ material, size, qty, chamfer, delivery }

### 状態管理
- draft（下書き）→ 保存時に自動でpending（回答待ち）に変更
- pending（回答待ち）/ done（完了）
- faxSent：FAX済みチェックボックス（boolean）

### 印刷レイアウト
- 1ページ最大7行・超えたら自動で2ページ目
- 最終ページのみ：備考欄・面取り位置参照図・お願い文言
- 面取り参照図：SVGで直方体に1〜12の番号（藤巻鋼材の書式準拠）

### 列構成（横A4）
No. / 材質 / 寸法（自由テキスト・折り返しあり）/ 数量 / 面取り指示 / 納期 / 単価記入欄
